# StarCode Studio – Payout Group System

## How to Use
1. Open `index.html` in Chrome or any modern browser.
2. Click **Send Robux** to open the payout dialog.
3. Type a Roblox username → real avatars will load from Roblox via CORS proxy.
4. Enter amount → Confirm → Done.

## API Fix
Uses `corsproxy.io` to bypass Roblox CORS restrictions. Requires internet connection.
